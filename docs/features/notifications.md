### Delayed Notifications

**Possible Causes**:
- Network delays
- Device battery saver mode
- Background app restrictions
- Server sync delays

**Solutions**:
1. Disable battery optimization for StackSave
2. Allow background data
3. Check network connection
4. Manually refresh app

### Duplicate Notifications

**Fix**:
1. Clear app cache
2. Disable and re-enable notifications
3. Update to latest version
4. Contact support if persists

## FAQ

**Q: Can I disable specific goal notifications?**
A: Not currently, but this feature is planned.

**Q: How long are notifications stored?**
A: In-app: 90 days. Email: According to your email provider.

**Q: Can I get notifications on multiple devices?**
A: Yes, all devices logged into your wallet will receive notifications.

**Q: Do notifications work offline?**
A: You'll receive pending notifications when you come back online.

**Q: Can I customize notification sounds?**
A: Yes, choose from preset sounds in notification settings.

**Q: Will I get notifications for test transactions?**
A: Yes, all transactions trigger notifications.
