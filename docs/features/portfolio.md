Your portfolio aggregates data from all your savings goals to give you insights into:

- Total savings across all goals
- Progress towards all targets
- Savings performance over time
- Goal distribution and allocation
- Historical trends and projections

## Troubleshooting

### Portfolio Not Updating

**Check**:
1. Internet connection
2. Pull to refresh
3. Wallet connection
4. Sync status

### Incorrect Totals

**Possible Causes**:
- Pending transactions
- Network delays
- Cache issues

**Solution**:
1. Refresh the app
2. Check on blockchain explorer
3. Clear app cache
4. Reconnect wallet

### Missing Transactions

**Verify**:
1. Transaction was confirmed
2. Correct network selected
3. Sufficient time for indexing
4. Check transaction hash on explorer

## FAQ

**Q: How often does the portfolio update?**
A: Real-time for on-chain data. Pull to refresh for latest values.

**Q: Can I hide completed goals?**
A: Yes, use filters or archive completed goals.

**Q: How far back does history go?**
A: Complete history from when you started using StackSave.

**Q: Can I share my portfolio?**
A: You can export data, but sharing features are coming soon.

**Q: Does portfolio tracking cost gas?**
A: No, viewing your portfolio is free. Only transactions cost gas.
